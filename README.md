# Assistant route

Aide à la modification d'attributs routiers.



Prérequis : Le "plugin\_maitre" doit obligatoirement être installé.
C'est ce plugin qui gère l'intégration dans le menu et/ou dans les barres d'outils.
Lien vers le plugin maître : [maitre-qgis-plugin sur GitHub](https://github.com/IGNF/maitre-qgis-plugin)



Pour utiliser les fonctionnalités de "sens de numérisation" il faut installer le plugin :
[Sens de numérisation](https://github.com/IGNF/sens_numerisation-qgis-plugin)



Pour utiliser les fonctionnalités de "Itinéraire le plus court" il faut installer le plugin :
[Chemin le plus court](https://github.com/IGNF/chemin-le-plus-court-qgis-plugin)

Lien vers la documentation : https://github.com/IGNF/assistant-route-qgis-plugin/blob/main/contribution%20directe%20(route).pdf
